# 設計書: page-tree

## 1. 目的
指定されたルートURLから再帰的にリンクを辿り、見つかったすべてのURLを抽出・リスト化するCLIツール。

## 2. 要件分析
- **再帰的クロール**: 1つ以上のルートURLから開始し、再帰的にページを訪問する。
- **境界設定**: ルートURLの配下（前方一致）にあるページのみを訪問対象とする。
- **リンク収集**: 訪問したページで見つかったすべてのリンク（別ドメインやルート外のリンクを含む）を収集する。
- **外部リンク**: 外部ドメインへのリンクは結果に含めるが、その先への再帰的なクロールは行わない。
- **Robots.txt**: `robots.txt` のルールおよび `Crawl-delay` を遵守する。
- **パフォーマンス**: 高速化のため、`asyncio` を用いた非同期リクエストを行う。
- **CLI**: `click` ライブラリを使用して実装する。

## 3. アーキテクチャ

### 3.1 コンポーネント
- **CLI (cli.py)**: エントリポイント。ユーザー入力と設定のハンドリング。
- **Crawler (crawler.py)**: 非同期クロールプロセス全体のオーケストレーション。
- **Scanner (scanner.py)**: 個別のページの取得とリンクの抽出。
- **Robots Manager (robots.py)**: `robots.txt` の取得・解析とアクセス許可判定。
- **URL Filter (filter.py)**: 訪問すべきURLか、単に収集するだけのURLかを判定するロジック。
- **Reporter (reporter.py)**: 出力形式の整形と保存。

### 3.2 データフロー
1. CLIがルートURLとオプションを受け取る。
2. ルートURLのドメインごとに `robots.txt` を取得。
3. ルートURLを「訪問予定キュー」に追加。
4. クローラーがキューからURLを取り出す:
   - 訪問済みかチェック。
   - `robots.txt` で許可されているかチェック。
   - ルートURLの配下かチェック。
   - 非同期でページ内容を取得。
   - すべてのリンクを抽出。
   - 各リンクについて:
     - 結果リストに追加。
     - 「ルートURL配下」かつ「同一ドメイン」であれば、訪問予定キューに追加。
5. キューが空になるか制限に達したら、結果を出力。

## 4. CLIオプション（案）
- `ROOT_URLS...`: 開始点となる1つ以上のルートURL。
- `-d, --max-depth INTEGER`: 最大再帰深度（デフォルト: 無制限）。
- `-c, --concurrency INTEGER`: 同時リクエスト数（デフォルト: 5）。
- `-o, --output PATH`: 結果を保存するファイルパス（未指定の場合はコンソールに出力）。
- `-f, --format [text|json|csv]`: 出力形式（デフォルト: text）。
- `-u, --user-agent TEXT`: カスタムUser-Agent文字列。
- `-t, --timeout FLOAT`: リクエストのタイムアウト秒数（デフォルト: 10.0）。
- `--delay FLOAT`: 同一ドメインへのリクエスト間の最小遅延時間。
- `--ignore-robots`: `robots.txt` を無視する（デフォルト: False）。
- `--include REGEX`: 収集・クロール対象に含めるURLの正規表現。
- `--exclude REGEX`: 除外するURLの正規表現。
- `--grouped`: リンクを「どのページで見つかったか」でグループ化して出力。

## 5. 技術スタック
- **言語**: Python 3.14+
- **HTTPクライアント**: `httpx` (async)
- **HTML解析**: `BeautifulSoup4` (標準の `html.parser` 使用)
- **CLIフレームワーク**: `click`
- **非同期処理**: `asyncio`

## 6. 課題と解決策
- **無限ループ防止**: URLを正規化し、訪問済みURLを `set` で管理。
- **レート制限**: セマフォによる全体的な並列数制御と、`robots.txt` に基づくドメインごとの遅延。
- **大規模サイト**: CLIツールとしてメモリ内管理で十分と想定するが、必要に応じて結果を逐次書き出す。
